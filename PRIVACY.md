# 개인정보 처리방침 (Privacy Policy)

**Easy PDF Tool**은 사용자의 개인정보를 중요하게 생각하며, 개인정보 보호법 및 관련 규정을 준수합니다.

## 1. 개인정보 수집 및 이용
* **Easy PDF Tool**은 어떠한 형태의 사용자 개인정보(이름, 이메일, 연락처, 기기 식별자 등)도 **수집, 저장, 공유, 전송하지 않습니다.**
* 본 애플리케이션은 사용자 본인의 로컬 PC 내에서 모든 데이터 처리(PDF 변환, 결합, 압축 등)를 수행하는 오프라인 전용 데스크톱 애플리케이션입니다.

## 2. 파일 및 문서 보안
* 사용자가 본 애플리케이션에 입력하거나 가공하는 모든 PDF 문서 및 파일은 **외부 서버로 업로드되지 않으며, 전적으로 로컬 컴퓨터 내에서만 안전하게 보관 및 처리**됩니다.
* 어떠한 상황에서도 네트워크를 통해 사용자의 문서 데이터가 제3자에게 노출되거나 외부로 유출되지 않습니다.

## 3. 네트워크 통신
* 본 애플리케이션은 외부 통신을 수행하지 않으며, 오프라인 환경에서도 모든 기능을 동일하게 제공합니다.

## 4. 문의 사항
* 개인정보 처리와 관련된 문의 사항이 있으실 경우, 본 GitHub 리포지토리의 Issue 등을 통해 문의해 주시기 바랍니다.

# Privacy Policy

**Easy PDF Tool** values your privacy and is committed to protecting your personal data.

## 1. Collection and Use of Personal Information
* **Easy PDF Tool** **does not collect, store, share, or transmit** any form of personal information (such as name, email, contact info, or device identifiers).
* This application is a strictly offline desktop utility. All data processing (PDF conversion, merging, compression, etc.) is performed locally on your personal computer.

## 2. Document and File Security
* Any PDF documents or files processed using this application are **never uploaded to external servers**. They are stored and handled entirely on your local machine.
* Under no circumstances is your document data exposed to third parties or leaked over the network.

## 3. Network Communication
* This application does not perform any outbound network communication and remains fully functional in offline environments.

## 4. Contact
* For inquiries regarding this policy, please contact us via the GitHub Repository Issues section.
